# Digit Recognition - A Simple Neural Network

A small project implementing a neural network from scratch (no deep‑learning frameworks) for handwritten digit recognition, inspired by [3Blue1Brown's neural networks playlist](https://youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi).

## Requirements

- Python
- numpy

## Project layout

- data/       - dataset  
- network/    - core network implementation  
- notebooks/  - Jupyter notebooks for experimentation and visualization  
- train.py    - for training the model  
- evaluate.py - for inference  
- README.md
